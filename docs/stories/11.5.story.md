# Story 11.5: Dashboard & Simple Pages Migration

## Status
Done

## Story
**As an** admin user,
**I want** the Dashboard and Apps pages rebuilt with shadcn/ui components,
**so that** I see a clean, card-based overview of system statistics and content.

## Acceptance Criteria
1. Dashboard page rebuilt with shadcn Card components for statistics (total books, app builds, publishers, storage)
2. Dashboard dual-table layout (Books + Apps) using shadcn Table component
3. Tables support sorting by column headers
4. Publisher filtering dropdown on dashboard
5. Soft delete with confirmation dialog on dashboard items
6. Auto-refresh mechanism preserved
7. Apps page rebuilt (wrapper for StandaloneAppsManager)
8. StandaloneAppsManager component migrated - template table, upload dialog, download/delete actions
9. All existing functionality preserved

### Dashboard Layout Structure
```

  <!-- Stats Grid -->

    <Card>stat 1</Card>
    <Card>stat 2</Card>
    <Card>stat 3</Card>
    <Card>stat 4</Card>
  </div>

  <!-- Filter Bar -->
  <div class="flex items-center gap-4">
    <Select>Publisher filter</Select>
  </div>

  <!-- Tables Grid -->
  <div class="grid grid-cols-1 gap-6 lg:grid-cols-2">
    <Card><Table>Books</Table></Card>
    <Card><Table>Apps</Table></Card>
  </div>
</div>
```

| TableSortLabel | Custom header with ChevronUp/ChevronDown icons |
| CircularProgress | Lucide Loader2 icon with animate-spin |
| Tooltip | Tooltip component from 11.2 |
