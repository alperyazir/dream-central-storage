4. All polling/auto-refresh mechanisms preserved
5. All form state management and save logic preserved
6. Loading states and error handling preserved

### Auto-refresh Pattern
```tsx
useEffect(() => {
  if (!hasActiveJobs) return;
  const interval = setInterval(fetchData, 10000);
  return () => clearInterval(interval);
}, [hasActiveJobs]);
```

### Autocomplete/Combobox for Bundles
If a proper Combobox is needed for book selection, use the shadcn pattern:
```tsx
<Popover>
  <PopoverTrigger asChild>
    <Button variant="outline" role="combobox">
      {selectedBook || "Select book..."}
    </Button>
  </PopoverTrigger>
  <PopoverContent>
    <Command>
      <CommandInput placeholder="Search books..." />
      <CommandList>
        <CommandItem>Book 1</CommandItem>
      </CommandList>
    </Command>
  </PopoverContent>
</Popover>
```
This requires creating `src/components/ui/command.tsx` using cmdk library.
